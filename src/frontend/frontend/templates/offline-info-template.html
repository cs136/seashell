Offline Mode

Seashell's offline mode allows you to write code and run C programs while temporarily disconnected from the internet. It stores all of your code locally using your web browser and synchronizes the local copy with the server when it becomes available.

If you want to enable or disable offline mode, you can do so at any time from Seashell's settings menu.

Note: Since Seashell's offline mode stores your assignment solutions locally, you should only enable offline mode on your own personal computer.

Browser Compatibility: We support offline mode in the following browsers: Firefox, Google Chrome, and Safari 10+.
